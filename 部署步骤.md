## 🔧 第三步：部署后端到Railway

1. 访问 https://railway.app
2. 使用GitHub账号登录
3. 点击 **"New Project"**
4. 选择 **"Deploy from GitHub repo"**
5. 选择你的仓库
6. Railway会自动检测并部署（等待2-3分钟）
7. 部署完成后，点击项目 → **"Settings"** → **"Domains"**
8. **复制生成的URL**（例如：`https://btc-backtest-api.up.railway.app`）
9. **把这个URL发给我**，我会更新前端代码

---

## 🎨 第四步：更新前端API地址

我会帮你更新 `index.html` 中的API地址，然后推送更新。

---

## 🌐 第五步：部署前端到Vercel

1. 访问 https://vercel.com
2. 使用GitHub账号登录
3. 点击 **"Add New Project"**
4. 选择你的仓库
5. Framework Preset 选择 **"Other"**
6. 点击 **"Deploy"**
7. 等待部署完成（约1分钟）
8. 获得前端URL（例如：`https://btc-backtest.vercel.app`）
