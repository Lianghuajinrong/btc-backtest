## 📦 第二步：部署后端到Railway（约5分钟）

### 快速步骤：

1. **打开浏览器，访问**：https://railway.app
2. **登录**：点击 "Login" → "Login with GitHub" → 授权
3. **创建项目**：点击 "New Project" → "Deploy from GitHub repo"
4. **选择仓库**：找到并选择 `Lianghuajinrong/btc-backtest`
5. **等待部署**：约2-3分钟，看到 "Deploy successful"
6. **获取URL**：点击项目 → "Settings" → "Domains" → 复制URL

## 🌐 第四步：部署前端到Vercel（约3分钟）

### 快速步骤：

1. **打开浏览器，访问**：https://vercel.com
2. **登录**：点击 "Sign Up" → "Continue with GitHub" → 授权
3. **导入项目**：点击 "Add New Project" → 选择 `Lianghuajinrong/btc-backtest`
4. **配置**：
   - Framework Preset: **"Other"**
   - 其他选项留空
5. **环境变量**（重要！）：
   - 点击 "Environment Variables"
   - Name: `API_BASE_URL`
   - Value: 你的Railway后端URL
   - 点击 "Add"
6. **部署**：点击 "Deploy" → 等待1-2分钟
7. **完成**：获得前端URL，访问测试
